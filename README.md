# PetLog 🐾 — 반려동물 라이프스타일 SNS

반려동물이 주인공인 감성 SNS. 보호자 계정 아래 펫 프로필을 여러 개 등록하고,
펫 프로필 단위로 게시물 작성 · 팔로우 · 좋아요 · 댓글 활동을 합니다.

## 기술 스택

- React 19 + Vite 8 + MUI 9 (모바일 웹 480px 프레임, 다크모드 자동 대응)
- Supabase (Auth + Postgres, `petlog_` 접두사 테이블 + RLS)
- 배포: GitHub Pages + GitHub Actions

## 1차 개발(MVP) 범위

- 이메일 회원가입 · 로그인 (만 14세 미만 가입 차단) + 펫 프로필 온보딩
- 메인 피드: 팔로잉 / 추천(전체 최신) 탭, 무한 스크롤, 더블탭 좋아요
- 게시물 작성 · 수정 · 삭제 (이미지 URL 최대 10장 + 본문 500자 + 해시태그 + 위치)
- 게시물 상세: 댓글 + 답글(1단계) + 이모지 반응, 공유(링크 복사)
- 탐색: 전체 최신 게시물 3열 그리드 (첫 칸 2배 강조)
- 마이페이지: 펫 프로필 카드 · 통계 · 게시물 그리드 · 펫 전환/추가, 생일 D-day
- 알림: 빈 상태 화면 (기능은 2차 개발)

## 실행

```bash
npm install
npm run dev
```

`.env` 파일에 Supabase 키가 필요합니다:

```
VITE_SUPABASE_URL=...
VITE_SUPABASE_ANON_KEY=...
```
